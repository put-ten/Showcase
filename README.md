# 页面展示 · AI Agent 驱动的前端成果集

> 通过 AI Agent 长链推理与多阶段协作，从零构建生产级前端页面。
> 单文件 HTML，浏览器直接打开。

---

## 核心逻辑流

本项目的构建采用 **多阶段长链推理 + 多 Agent 协作** 架构：

1. **需求理解与方向收敛** — Brainstorming Agent 与用户对话式交互，明确视觉风格与约束
2. **设计决策** — 多候选方案中快速决策，无需翻看大量参考图
3. **全栈实现** — 单 Agent 独立完成完整页面，含 Three.js 3D 场景、粒子系统、鼠标绕流交互等
4. **即时交付** — 浏览器直接打开，用户即时预览

每次构建包含 20+ 轮推理决策（从整体布局到每个像素的动画曲线），全部由 AI Agent 独立完成。

## 技术亮点

- **单文件交付** — 纯 HTML 内联 CSS/JS，无构建工具
- **Three.js 3D 星系** — 3 条螺旋臂 × 360 粒子 + 200 星尘 + 600 背景星
- **鼠标绕流** — Raycaster 投射 + 实时粒子位置更新，鼠标推开粒子后平滑回弹
- **轨道控制** — OrbitControls 拖拽旋转 + 滚轮缩放 + 自动旋转
- **Canvas 纹理** — 粒子光晕由 Canvas 径向渐变生成，AdditiveBlending 叠加发光
